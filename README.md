# cognito-aws-nodejs
Integrated AWS Cognito with Node js

## Feature:

- Login
- Register
- Verify Code SignUp
- Refresh Token
- Verify Token
- Change Password
- Forgot Password
- Get User
- Update User

## Feature SignIn with Google and Manage Token

Dont Forget SetUp your configuration `.env`:
```
AWS_COGNITO_USER_POOL_ID=
AWS_COGNITO_CLIENT_ID=
AWS_COGNITO_REGION=
AWS_COGNITO_IDENTITY_POOL_ID=
AWS_ACCESS_KEYID=
AWS_SECRET_ACCESS=
AWS_DOMAIN_NAME=
AWS_REDIRECT_URL=
